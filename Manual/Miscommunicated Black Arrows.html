On the Subject of Miscommunicated Black Arrows

Point this away from being a boss module. Point it such that a boss arrows module would never exist.

If the arrows are not black, the arrows are flashing directions, or the display is empty, you are looking at a different module.

To disarm this module, complete 5 stages in a row using two two-digit numbers on the table provided on the next page. Start with concatenating the 3rd and 6th characters of the serial number as the first number and the displayed number as the second. Do this every time upon taking a strike on this module.

Take the number of ports on the bomb and add 1 to this. Count this many items in the list provided and assign this to the first number to obtain a position. Assign the next item underneath, wrapping around to the top if necessary, to the second number to obtain another position. Take the horizontal and vertical distances between those two values. Press the most optimal direction such that, when accounting for wrap-around, the user travels 1 step closer to the second position from the first position, by narrowing the maximum orthogonal distance.

When a stage is completed or when the module strikes, a new two-digit number will show. For each completed stage, the new displayed number will be assigned the item directly underneath the item used for the second number, wrapping around if necessary. The second position obtained will become the first position for this stage, repeating until the module is solved.

The two digit number is referred to AB on the list provided.

  • A as row, B as column
  • AB on the table
  • B matches last digit of values in A’s column
  • A as column, B as row
  • BA on the table
  • B matches first digit of values in A’s row
